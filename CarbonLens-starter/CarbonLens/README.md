## The Problem

- **Only 7%** of companies comprehensively measure emissions across all scopes — down from 10% (BCG 2025)
- **80-90%** of emissions sit in Scope 3 (supply chain) — nearly impossible to track
- Existing tools cost **$50K+/year** and are inaccessible to SMEs
- Corporate sustainability reports are often **unverifiable** against independent data

## What CarbonLens Does

### 🔎 Verify Mode — Sustainability Claims Detector
Enter a company name → Five AI agents cross-reference their sustainability report against EPA facility data, industry benchmarks, and public records → Get a **Transparency Score (0-100)** with cited evidence.

### 📊 Measure Mode — Scope 3 Calculator
Upload procurement data (CSV/Excel) → Four AI agents parse, classify, and calculate emissions using verified EPA and DEFRA emission factors → Get a complete **Scope 3 breakdown** with ranked suppliers and reduction recommendations.

### Anti-Hallucination Design
- **LLMs** handle understanding: parsing documents, classifying categories, cross-referencing claims
- **Code** handles math: emission factor lookups, calculations, aggregation
- **No emission number is ever generated by an LLM** — every number traces to a published database + explicit formula

## Data Sources

| Source | Type | Use |
|--------|------|-----|
| EPA EEIO | Spend-based emission factors (~1000 sectors) | Measure mode + estimation engine |
| UK DEFRA | Activity-based conversion factors | Measure mode (when quantity data available) |
| EPA GHGRP | Facility-level emissions (8000+ US facilities) | Verify mode cross-referencing |
| EPA ECHO | Environmental enforcement records | Verify mode violation checking |
| Industry benchmarks | Sector-average emission intensities | Verify mode estimation |

## Getting Started

### Prerequisites
- Python 3.11+
- Node.js 18+
- Anthropic API key

### Installation

```bash
# Clone the repository
git clone https://github.com/RastinAghighi/CarbonLens.git
cd CarbonLens

# Backend setup
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Initialize the emission factor database
python scripts/init_database.py

# Set environment variables
export ANTHROPIC_API_KEY=your_key_here

# Start backend
uvicorn main:app --reload

# Frontend setup (new terminal)
cd frontend
npm install
npm run dev
```

### Environment Variables
```
ANTHROPIC_API_KEY=       # Required - Claude API key
DATABASE_PATH=           # Optional - defaults to ./data/carbonlens.db
```
